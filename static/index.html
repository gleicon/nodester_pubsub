PubSub

PubSub for Nodester

PubSub message bus for nodester based on HTTP. Send messages thru POST requests, follow up with a GET request that binds to a COMET endpoint. Routes are created dynamically. You can even bind to a route that never received messages.

Quickstart

open a terminal and type: 

 curl http://pubsub.nodester.com/lmao

open a second term and type: 

curl -d 'body=sup' http://pubsub.nodester.com/lmao

watch first terminal in amazement. Bear in mind that depending on the message size, it can take some time to be rendered on a browser, but will be retrieved by javascript.
